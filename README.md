# Tawuri Tifinagh Serif

![font preview](https://assets.bensadik.net/CnP_19032026_104643.png)

A modified version of [IBM Plex Serif](https://github.com/IBM/plex) extended with Tifinagh script support and Latin characters for Tamazight romanization.

## About

Tawuri Tifinagh Serif brings the elegance of IBM Plex Serif to the Amazigh writing system. This project adds a full set of Neo-Tifinagh glyphs alongside the special Latin characters commonly used to romanize Tamazight, making it suitable for bilingual typesetting and digital content in both scripts.

## Supported Characters

### Tifinagh (Neo-Tifinagh)

ⴰ ⴱ ⴳ ⴷ ⴹ ⴻ ⴼ ⴽ ⵀ ⵃ ⵄ ⵅ ⵇ ⵉ ⵊ ⵍ ⵎ ⵏ ⵓ ⵔ ⵕ ⵖ ⵙ ⵚ ⵛ ⵜ ⵟ ⵡ ⵢ ⵣ ⵥ ⵯ

### Latin Extensions for Tamazight Romanization

Lowercase: ḍ ḥ ɛ ṛ ṣ ɣ ṭ ẓ ʷ Uppercase: Ḍ Ḥ Ɛ Ṛ Ṣ Ɣ Ṭ Ẓ ʷ

### Inherited from IBM Plex Serif

All original Latin, Cyrillic, and Greek character sets from IBM Plex Serif are fully preserved.
